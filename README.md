# NASM Assembly Language Labs

This repository contains selected laboratory works written in x86 assembly language (NASM).

## Covered topics
- string processing
- registers and loops
- stack and command-line arguments
- Linux system calls
- file operations
- UNIX time handling
- executing external programs (execve)

## Environment
- NASM
- Linux (x86, 32-bit)
- SASM IDE / Online Assembly

## Notes
These works demonstrate progression from basic assembly concepts to system-level programming.
